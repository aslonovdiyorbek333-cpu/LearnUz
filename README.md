## 🚀 O'rnatish

```bash
# 1️⃣ Loyihani klonlash
git clone https://github.com/aslonovdiyorbek333-cpu/LearnUz.git
cd LearnUz

# 2️⃣ Virtual muhit yaratish va yoqish
python -m venv venv
venv\Scripts\activate        # Windows
source venv/bin/activate     # Mac / Linux

# 3️⃣ Kutubxonalarni o'rnatish
pip install -r requirements.txt

# 4️⃣ Ma'lumotlar bazasini sozlash
python manage.py migrate

# 5️⃣ Superuser yaratish
python manage.py createsuperuser

# 6️⃣ Serverni ishga tushirish
python manage.py runserver
```

> 🌐 Brauzerda oching: **http://127.0.0.1:8000**
