# 🎓 LearnUz — Online Ta'lim Platformasi

O'zbekistondagi eng yaxshi online ta'lim platformasi. Istalgan vaqt, istalgan joydan o'qing!

## 🚀 Loyiha haqida

**LearnUz** — Django asosida qurilgan fullstack web dastur. Foydalanuvchilar kurslarga yozilishi, darslarni ko'rishi va o'z progressini kuzatishi mumkin.

## ⚙️ Texnologiyalar

- **Backend:** Python 3.13, Django 6.0
- **Frontend:** HTML5, CSS3, Bootstrap 5
- **Ma'lumotlar bazasi:** SQLite
- **Versiya nazorati:** Git & GitHub

## 📦 Xususiyatlar

- ✅ Foydalanuvchi ro'yxatdan o'tish va kirish
- ✅ Kurslar ro'yxati va kategoriyalar
- ✅ Kurs batafsil sahifasi
- ✅ Darslarni ko'rish
- ✅ Kursga yozilish
- ✅ Foydalanuvchi profili
- ✅ Admin panel
- ✅ Xavfsizlik validatsiyasi

## 🛠️ O'rnatish

```bash
# 1. Loyihani klonlash
git clone https://github.com/aslonovdiyorbek333-cpu/LearnUz.git
cd LearnUz

# 2. Virtual muhit yaratish
python -m venv venv
source venv/bin/activate  # Windows: venv\Scripts\activate

# 3. Kutubxonalarni o'rnatish
pip install -r requirements.txt

# 4. Migratsiyalarni bajarish
python manage.py migrate

# 5. Suprefoydalanuvchi yaratish
python manage.py createsuperuser

# 6. Serverni ishga tushirish
python manage.py runserver
```
